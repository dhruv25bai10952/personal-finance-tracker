## Problem Statement
Most individuals do not regularly track their assets, liabilities, and daily expenses. This leads to poor financial decisions and surprise shortages. This project solves this real-world problem by providing a simple, menu-driven personal finance tracker that calculates live net worth.

## Objectives
- Add and manage assets, liabilities, and expenses
- Persist all data using CSV files
- Generate detailed financial reports including net worth
- Use only standard Python libraries (no external packages)

## Features
- Menu-driven console interface
- Data persistence across sessions
- Complete financial summary with pie-chart-like breakdown (text)
- Rule-based expense categorization

## Tech Stack
- Python 3.11+ (standard libraries only)
